# Примерная структура папки с решениями Линейной Алгебры
```
LinAlg/
├── README.md
├── Stringify.py (функция для приведения np.array к Geolin виду)
├── ... (прочие не нужные особо файлы, можете покопаться если интересно)
├── HW/
│   ├── pdf_tasks/ (папка с pdf файлами заданий; ищите решение конкретно того или иного задания с помощью ctrl+F по юпитер ноутбуку и вбивайте туда условия заданий из папки pdf_tasks)
│   ├── <NUMBER>.ipynb (файлы с решениями заданий из данной домашки; i.e. fifth.ipynb, sixth.ipynb)
├── PT's/ (папка с контрольными работами)
```
